Gets a file from remote

Usage:
  rospo get [user@]host[:port] remote [local] [flags]

Examples:

  # downloads a file from the remote server
  $ rospo get myserver:2222 file.txt .

  # downloads recursively all contents of myremotefolder to local current working directory
  $ rospo get myserver:2222 /home/myuser/myremotefolder -r

  # downloads recursively all contents of myremotefolder to local target directory
  $ rospo get myserver:2222 /home/myserver/myremotefolder ~/mylocalfolder -r
	

Flags:
  -c, --concurrent-downloads int   concurrent downloads (recursive only) (default 4)
  -b, --disable-banner             if set disable server banner printing
  -h, --help                       help for get
  -i, --insecure                   disable known_hosts key server verification
  -j, --jump-host string           optional jump host conf
  -k, --known-hosts string         the known_hosts file absolute path (default "/Users/ferama/.ssh/known_hosts")
  -w, --max-workers int            parallel workers per file (default 12)
  -p, --password string            the ssh client password
  -r, --recursive                  if the copy should be recursive
  -s, --user-identity string       the ssh identity (private) key absolute path (default "/Users/ferama/.ssh/id_rsa")

Global Flags:
  -q, --quiet   if set disable all logs
